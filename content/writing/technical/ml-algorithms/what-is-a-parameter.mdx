## The Simplest Way to Understand It

A parameter is just a number the model learns.

Think of it like a knob.

The model keeps adjusting these knobs until it starts making better predictions.

## Example

If you had a simple model:

y = wx + b

Then:
- w is a parameter
- b is also a parameter

The model learns both.
